# Changelog

Track the evolution of LCSoft.Pyxenv - from new features and improvements to breaking changes and bug fixes. This helps you understand what's changed between versions and plan your upgrades.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-10-25
### Added
- Update of `LCSoft.Pyxenv` with:
  - ~80% unit test coverage.
  - All script to generate build (wheel files)
  - Script to generate exe and .msi files
  - Apply best practices and SoC
  - Add Sphinx documentation
- Included GitHub Actions to auto-publish to NuGet.
- Added debug project for testing and experimentation.
- Configured test and CI workflow.
