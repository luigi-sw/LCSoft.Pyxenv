Installation
============

Requirements
------------
* Python â‰¥ 3.8
* Windows, Linux or macOS

From PyPI (stable)
------------------
.. code-block:: console

   $ pip install pypx

From source (development)
-------------------------
.. code-block:: console

   $ git clone https://github.com/seu-usuario/pypx.git
   $ cd pypx
   $ python -m venv venv
   $ source venv/bin/activate      # Windows: venv\Scripts\activate
   $ pip install -e .[dev]

Verify
------
.. code-block:: console

   $ pypx --help
