Changelog
=========

v0.2.0 (2025-06-XX)
-------------------
* Added ``pypx install`` and ``uninstall`` commands.  
* Automatic PATH management on Windows.  
* Improved error messages for missing downloads.  

v0.1.0 (2025-06-01)
-------------------
* Initial release.  
* ``list``, ``venv create`` and ``venv remove`` sub-commands.  
* Cross-platform support planned (Windows, Linux, macOS).
